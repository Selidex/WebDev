HTML

Intro

HTML (Hypertext Markup Language) form the building blocks of our website. The vast majority of our content on a webpage is created using HTML, to later be styled with css, and possibly controlled with js. HTML is made up of tags that tell a web browser how to display the content of the web page, for example, this part right here is contained within a <p> tag. As of HTML 5.2 there are 142 HTML tags, although only 115 tags are deemed functional. This website is a work in progress, and will start with basic html tags that are commonly used. Other tags will be added as time goes on.

Tag Format

The standard format for a tag is as follows:

<tagname>Content goes here</tagname>

Notice the forward slash ( / ) before tag name in the second set of angular braces ( < > ). This is called a closing tag, and most HTML tags require both an opening and closing tag. There are examples of tags that do not require a closing tag. These tags are commonly called empty tags, void tags, or self-closing tags. Tags can also have attributes that edit how the tag behaves. When a tag possesses such an attribute it will be included in the opening tag's angular braces, for example to add controls to an audio tag we use the following tag format:

<audio controls><audio>
