Web Development

Purpose

This website is being made in service to help people make their own websites using basic html, css, and javascript. Currently a work in progress, the website will be periodically updated to include more information.

Tabs

  • Home - Returns the user to this page.
  • Deploy - Takes the user to a page displaying step by step instructions to deploy a website from Codepen to Github pages.
  • HTML - Takes the user to a page with descriptions for commonly used html tags
  • CSS - Takes the user to a page with descriptions for commonly used CSS attributes.
  • JS - Takes the user to a page with some basic javascript examples

Alternate Resources

There are other great sites you can use both to learn html/css/js, and to serve as a refresher.

  • Sololearn: A website with lesson based learning for html, css, js, and more.
  • W3SchoolsA website with easy to understand tutorials, as well as "Try it Yourself" self contained testing to practice what you learn.
